# Data Warehouse and Analytics Projects.

Welcome to my **Data Warehouse and Analytics Project**.!!
This repository showcases how raw business data can be transformed into meaningful insights through modern data analytics and business analytics practices. It's design as a portfolio project to highlight both technnical skills and problem - solving ability in real - world scenarios.

# Project Overview.

## Building the Data Warehouse.!
### Objective
Design and implement a scalable data warehouse using SQL server to consolidate sales data from multiple systems, ensuring a single source of truth for reporting and decision - making.

### Key Highlights:
**Data Source:** Integrated ERP and CRM datasets(CSV Files).
**Data Quality:** Applied cleansing techniques to handle duplicates, missing values, and inconsistencies.
**Integration:** Unified both sources into a clean, business - friendly schema optimized for analytical queries.
**Scope:** Focused on the latest dataset for simplicity and relevance.
**Documentation:** Delivered clear data model diagrams and explanations for both technical and non - technical stakeholders.

# BI: Analytics & Reporting.!
## Objective:
Develop SQL‑based analytics that empower stakeholders with actionable insights.

### Insights Delivered:
**Customer Behavior:** Patterns in engagement, retention, and purchasing.
**Product Performance:** Identifying top‑performing products and areas needing improvement.
**Sales Trends:** Tracking growth, seasonality, and forecasting opportuniti.
